# Specialist headshots

Place headshot photographs here using the slugs defined in `src/content/especialistas-page.ts`:

- `diretora-clinica.jpg`
- `implantodontia.jpg`
- `ortodontia.jpg`
- `periodontia.jpg`
- `odontopediatria.jpg`
- `estetica.jpg`

Specs:
- Aspect ratio: 4:5 portrait (matches Treatment cards)
- Minimum: 800 × 1000 (1600 × 2000 preferred for retina)
- Format: optimized JPG, target < 250 KB per image
- Composition: shoulders and head, slight off-center is fine, neutral or softly textured background
- Lighting: warm, natural — never clinical fluorescent. The Biodonth marmorino-on-bronze palette favors warmer tones.
- Dress: smart but not "lab-coat-and-stethoscope clinical." We're a luxury wellness clinic, not a hospital.

Until real photographs are placed here, each EspecialistaCard renders a BRANDED placeholder showing the specialist's initials on a bronze background in Instrument Serif. This is intentional — it looks like a brand element, not a "missing image."
