# Hero imagery

Place the homepage hero photograph at:

`hero-smile.jpg`

Recommended specs:
- Aspect ratio: 4:3 landscape (or close — 16:10 also works)
- Minimum dimensions: 1600 × 1200
- Format: optimized JPG (or WebP if served via next/image automatically)
- Subject: a real Biodonth patient or natural-feeling stock with a candid smile, soft warm light, neutral background. NO clinical zoom shots, NO white-teeth-blast stock.
- Color tone: warm, slightly desaturated, harmonizes with bronze + concrete brand palette

Until a real photograph is placed here, the Hero component will fall back to a placeholder Unsplash image (defined in `src/content/home.ts`).
